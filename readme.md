# JavaScript Snippets for Acode

A collection of JavaScript snippets for the Acode editor.

## How to Use

1.  Open a JavaScript file (`.js`, `.jsx`, `.ts`, `.tsx`).
2.  Type the prefix of the snippet you want to use.
3.  The snippet will appear in the autocomplete suggestions.
4.  Select the snippet to insert it into your code.

## Available Snippets

| Prefix | Description                        |
| :----- | :--------------------------------- |
| `ejj`  | Arrow function with default export |
| `ii`   | Import default                     |
| `arr`  | Arrow function                     |
